Що вміє програма
Інтерактивне TUI-меню: керування стрілочками на клавіатурі, вибір через Enter.

Автопошук тестів: програма сама сканує папку TestData та виводить список доступних тем (ігнорує пусті файли-привиди).

Перемішування: питання та варіанти відповідей перемішуються випадковим чином при кожному запуску.

Анти-пробіл (Захист від багів): стоїть жорстка валідація. Якщо натиснути три пробіли замість питання чи відповіді — програма видасть помилку і попросить ввести нормальний текст.

Керування темами: можна додавати нові відкриті питання, видаляти окремі питання або повністю зносити файл теми прямо з консолі.

Статистика та Історія: результати зберігаються у файл history.txt, програма рахує загальний відсоток успішності.

Запустити програму: dotnet run ,  збудувати і запустити через IDE або завантажити готовий .exe файл.

Запуск тестів : dotnet test для запуску юніт-тестів, які перевіряють основні функції програми. Тести можна запустити так само як і програму через IDE.
